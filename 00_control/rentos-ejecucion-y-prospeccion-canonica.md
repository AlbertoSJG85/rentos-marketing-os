## 2. Flujo comercial mínimo

**Outreach → Interés → Demo / visualización → Onboarding → Activación → Feedback → Decisión**

### Métrica reina
**Conectar al menos 1 propiedad en los primeros 5 días.**

### Señales a vigilar
- respuesta al outreach,
- leads que piden demo,
- onboarding completado,
- primera propiedad conectada,
- uso en semana 2,
- frases literales de valor percibido.

## 3. Pipeline operativo

| Estado | Significado |
|---|---|
| Identificado | Lead encontrado con datos básicos. |
| Priorizado | Lead con encaje alto para empezar outreach. |
| Preparado | Mensaje personalizado listo. |
| Contactado | Outreach enviado. |
| Demo / Interés | Respondió o pidió ver más. |
| Activado | Se registró y conectó propiedad. |
| No encaja / Churn | Descarta o pierde interés. |

## 6. Cómo usar este archivo en la práctica

Para cada lead solo hace falta añadir, en una nueva línea de trabajo o en una tabla auxiliar:
- fecha de primer contacto,
- canal usado,
- copy usado,
- respuesta,
- siguiente paso.

## 7. Siguientes pasos exactos

1. Elegir 8 leads del bloque caliente.
2. Personalizar el primer mensaje con zona y encaje.
3. Lanzar outreach inicial.
4. Registrar respuesta en este mismo archivo o en tu CRM.
5. Empujar demo corta o prueba.
6. Medir quién entra a onboarding.
7. Hablar con activados y recoger feedback.
